# Acerca de mi

Data Analyst Junior bilingüe con sólidas habilidades en Power BI y bases de datos SQL, con enfoque en transformar datos en insights claros y accionables con la creación de dashboards interactivos. Creativa, adaptable y responsable. Con experiencia en entornos de call center, enfocada en la optimización y creación de nuevos reportes usando herramientas como Excel o Power BI. 


Tuve una transición desde atención al cliente, donde trabajé más de 8 años resolviendo problemas complejos de clientes y mejorando procesos internos. Durante el bootcamp de Data Analyst en TripleTen, aprendí a convertir datos en insights claros y accionables utilizando SQL y Python.



Disfruto limpiando conjuntos de datos desordenados, encontrando patrones y explicando resultados de forma simple y visual para stakeholders no técnicos. Algunos de mis proyectos incluyen el análisis de abandono de clientes, la creación de un dashboard de desempeño de ventas y la identificación de cuellos de botella operativos para una empresa de e-commerce ficticia.



Actualmente busco un rol como Analista de Datos junior donde pueda combinar mis habilidades analíticas, experiencia en comunicación y enfoque orientado al cliente para ayudar a los equipos a tomar mejores decisiones.
